# Lezzet-i Âlâ

Restoranlar için modern dijital menü yönetim sistemi.

## Özellikler

- Restoran menüsü oluşturma ve yönetme
- QR kod ile müşteri erişimi
- Çoklu restoran desteği (süper yönetici)
- Gerçek zamanlı menü güncelleme
- Mobil uyumlu tasarım

## Teknoloji

- Next.js 14 (App Router)
- Supabase (Auth, Database, Storage)
- Tailwind CSS + shadcn/ui
- TypeScript

## Başlangıç

```sh
# Depoyu klonlayın
git clone <YOUR_GIT_URL>

# Proje dizinine gidin
cd <YOUR_PROJECT_NAME>

# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```
